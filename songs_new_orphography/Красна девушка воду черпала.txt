Красна девушка воду черпала;
Добрый молодец, он коня поил,
Он коня поил, подговаривал:
Ты пойдем, девица, в наш Казань-город!
Наш Казань-город на красе стоит,
На красе стоит, на угожестве;
Казань-реченька медом протекла,
Мелки ручейки — зеленым вином;
По гораи-горам по высокъ
Дежат камушки самоцветные;
По лужкам-лужкам по зеленыим
Растет трава все шелковая;
На траве цветут цветы лазоревы;
По лесам-лесам по дремучъ
Бежат куны заморск» —
— Не обманывай, добрый молодец!
Я во Казань-городе была рожена,
В Казань-реченьке была купана.
Ваш Казань-город на ноже стоит,
Казань-реченька кровью протекла,
Мелки ручейки — горючми слезьми;
По горам-горам по высокъ
Не каменья самоцветные —
Лежат головы человечески;
По лужкам-лужкам по зеленыим
Не трава растет шелковая —
Лежат волоса человечески;
По лесам-лесам по дремучъ
Не куны бежат заморскя —
Бежат волки едуч»
