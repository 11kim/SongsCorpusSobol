По горам, по горам,
Я по горам ходила,
Все цветы, все цветы,
Все цветы видела;
Нет цветка, нет цветка,
Нет цветка милаго,
Нет цветка милаго,
Моего любимаго!
По двору, по двору,
Я по двору ходила,
Всех коней, всех коней,
Всех коней видела;
Нет коня, нет коня,
Нет коня сиваго,
Сиваго, сиваго,
Моего любимаго!
По сеням, по сеням,
Я по сеням ходила,
Все шубы, все шубы,
Все шубы видела;
Нет шубы, нет шубы,
Нет шубы тисетовой!
По горнице, по горнице,
Я по горнице ходила,
Всех гостей, всех гостей,
Всех гостей я видела;
Нет гостя, нет гостя,
Нет гостя милаго,
Нет гостя милаго —
Батюшки родимаго.
Али мне, али мне
Послать было некого?
