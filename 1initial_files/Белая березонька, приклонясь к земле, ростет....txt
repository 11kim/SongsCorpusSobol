<h4 L=0 title="&lt;Текст&gt;" id=Текст></h4>
<p class=stih><b>Б</b><font class=old>&#x463;</font>лая березонька, приклонясь къ земл<font class=old>&#x463;</font>, ростетъ;<br>
На эту березоньку слет<font class=old>&#x463;</font>лися пташечки,<br>
Пташечки-пичужечки, горькія кукушечки.<br>
Кукуетъ кукушечка во своемъ тепломъ гн<font class=old>&#x463;</font>зд<font class=old>&#x463;</font>:<br>
«Св<font class=old>&#x463;</font>тъ, ты, мое гн<font class=old>&#x463;</font>здышко, совитое, теплое,<br>
Совитое, теплое, гн<font class=old>&#x463;</font>здо разореное!»<br>
Подъ этой березонькой собралися молодушки.<br>
Одна-то молодушка призадумавшись стоитъ,<br>
Призадумавшись стоитъ, при печали говоритъ:<br>
«У меня, молодушки, четыре заботушки:<br>
Первая заботушка — чужа дальняя сторонушка;<br>
Другая заботушка — свекоръ да свекровушка;<br>
Третія заботушка — деверь да золовушка;
</BLOCKQUOTE><BLOCKQUOTE class=page><P><p><span class=page id=$p391>- 391 -</span></P>
<p class=stih>Четверта заботушка — мужъ удалая головушка.<br>
Мужъ удала голова, чешетъ кудри завсегда;<br>
Онъ гуляетъ отъ меня по вс<font class=old>&#x463;</font>мъ селамъ-деревнямъ,<br>
По вс<font class=old>&#x463;</font>мъ селамъ-деревнямъ, по господскіимъ дворамъ,<br>
По господскіимъ дворамъ, по солдатскимъ по вдовамъ,<br>
По солдатскимъ по вдовамъ, по д<font class=old>&#x463;</font>вчонкамъ-сиротамъ,<br>
По д<font class=old>&#x463;</font>вчонкамъ-сиротамъ, по чужемужніимъ женамъ.»
<p class=stih>&nbsp;&nbsp;&nbsp;&nbsp;Д<font class=old>&#x463;</font>вчоночка-сиротинка звала его ночевать:<br>
«Приди, приди, Ванюшка, ночуй ночку у меня!» —<br>
— «Боюсь, боюсь, д<font class=old>&#x463;</font>вушка, просплю долго у тебя!» —<br>
— «Не бойсь, не бойсь, Ванюшка, я на зорьк<font class=old>&#x463;</font> разбужу,<br>
По утру ранымъ-ранехонько за ворота провожу,<br>
За ворота провожу, три словечушка скажу:<br>
Первое словечушко — хотятъ Ванюшку поймать,<br>
Другое словечушко — во жел<font class=old>&#x463;</font>зушки сковать,<br>
Третіе словечушко — во солдатушки отдать!<br>
Подуй, подуй, погодушка, занеси Ванины сл<font class=old>&#x463;</font>ды!»
<h4 L=0 title="&lt;Комментарий&gt;" id=Источник></h4>
