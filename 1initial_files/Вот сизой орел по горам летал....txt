<h4 L=0 title="&lt;Текст&gt;" id=Текст></h4>
<p class=stix3>Вотъ сизой орелъ по горамъ леталъ,<br>
По горамъ, по доламъ, тяжело вздыхалъ<br>
Тяжел<font class=old>&#x463;</font>й того д<font class=old>&#x463;</font>вка плакала,<br>
Не по батюшк<font class=old>&#x463;</font>, не по матушк<font class=old>&#x463;</font>, —<br>
По миломъ дружк<font class=old>&#x463;</font> по Иванушк<font class=old>&#x463;</font>.<br>
У Иванушки вились кудрюшки,<br>
Они вилися, разв<font class=old>&#x463;</font>валися,<br>
Въ три ряда, въ три ряда заплеталися,<br>
Во четвертый рядъ по плечамъ висятъ.<br>
Далеко я шла провожать дружка —<br>
Дальше города дальше Новаго.<br>
Посередъ Москвы разставалися.<br>
И вс<font class=old>&#x463;</font> люди дивовалися:<br>
Да и кто такой разстается съ к<font class=old>&#x463;</font>мъ?<br>
Либо братъ съ сестрой, либо мужъ съ женой?<br>
Либо первый другъ съ полюбовницей?
<p class=stix3k1>Какъ и по мосту-мосту по калиновому,<br>
По другому по сухому перелетывалъ,<br>
Тамъ и шелъ-прошелъ молодой почтарь,<br>
Молодой почтарь, голубой кафтанъ,<br>
Голубой съ галуной, весь изорванный худой;<br>
Полы машутся, раздуваются;<br>
Миткалевая рубашечка б<font class=old>&#x463;</font>л<font class=old>&#x463;</font>ется.<br>
Б<font class=old>&#x463;</font>ла, б<font class=old>&#x463;</font>ла, мн<font class=old>&#x463;</font> сударушка дала,<br>
Мн<font class=old>&#x463;</font> дала, дала, мн<font class=old>&#x463;</font> пожаловала.<br>
«Милъ, носи, носи, носи, а другую не проси!»
