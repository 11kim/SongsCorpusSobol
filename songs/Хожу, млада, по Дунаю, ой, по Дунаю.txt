Хожу, млада, по Дунаю, ой, по Дунаю,
Хожу, млада, по тихому, ой, по тихому.
Зайду, млада, во бесѣду, ой, во бесѣду,
Зайду, млада, во смиренну, ой, во смиренну.
Во смиренной сидитъ старый, ой, кашливый;
На колѣнахъ держитъ гусли, ой, лубяныя,
По гуслямъ-то струны водитъ, ой, лычаныя.

У меня, младой-младеньки, ой, сердце ноетъ!
Сердце ноетъ, унываетъ, ой, унываетъ!

Хожу, млада, по Дунаю, ой, по Дунаю,
Зайду, млада, во бесѣду, ой, во смиренну.
Во смиренной сидитъ малый, ой, сопливый;
На колѣнахъ держитъ гусли, ой, досчатыя,
По гуслямъ-то струны водитъ, ой, силянныя.
У меня, младой-младеньки, ой, сердце ноетъ,
Сердце ноетъ, унываетъ, ой, унываетъ!

Хожу, млада, по Дунаю, ой, по Дунаю,
Хожу, млада, по быстрому, ой, по быстрому,
Зайду, млада, во бесѣду, ой, во веселу.
Во веселой сидитъ младый, ой, мой милый;
На колѣнахъ держитъ гусли, ой, звончатыя,
По гуслямъ-то струны водитъ, ой, золотыя.
У меня, младой-младеньки, ой, сердце радо,
Сердце радо, взрадовалось, ой, взрадовалось,
Рѣзвы ножки заплясали, ой, заплясали,
Бѣлы ручки защелкали, ой, защелкали.
